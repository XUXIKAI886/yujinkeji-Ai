# 域锦科技 AI 助手平台

一个基于 AI 技术的智能助手平台，专注于提升外卖运营效率和数据分析能力。

## 功能特性

### 1. AI 智能对话
- 多场景专业 AI 助手
- 自然语言交互
- 智能问答和建议
- 个性化助手欢迎语
  - 美团全能客服：客服咨询指引
  - 外卖菜品图生成：图片生成参数指引
  - 抖音文案提取：链接格式说明
  - 外卖商圈调研：数据采集指引
  - 外卖店铺数据分析：数据导出指引
  - 外卖店铺诊断：诊断项目指引

### 2. 数据可视化应用
- 销售数据可视化
  - 销售趋势分析
  - 营业额统计
  - 客流量分析
  - 转化率追踪
  - 实时数据更新
- 商圈数据分析
  - 区域市场分析
  - 竞争态势评估
  - 商圈规模统计
  - 市场机会识别
- 竞店数据分析
  - 竞争对手监控
  - 价格策略分析
  - 销量对比
  - 市场份额评估

### 3. 用户管理系统
- 基于邀请码的注册机制
  - 管理员生成8位邀请码
  - 7天有效期限制
  - 一次性使用限制
  - 邀请码状态追踪
- 增强的密码安全策略
  - 最小长度：8字符
  - 必须包含大小写字母
  - 必须包含数字
  - 必须包含特殊字符
  - 实时密码强度验证

## 安装部署

### 环境要求
- Node.js >= 14.0.0
- MongoDB >= 4.0
- NPM 或 Yarn

### 安装步骤

1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
# 安装后端依赖
npm install

# 安装前端依赖
cd client
npm install
```

3. 配置环境变量
```bash
# 后端配置 (server/.env)
PORT=3000
MONGODB_URI=mongodb://localhost:27017/your_database
JWT_SECRET=your_jwt_secret
API_VERSION=1.0.0

# 前端配置 (client/.env)
REACT_APP_API_URL=http://localhost:3000/api
```

4. 启动服务
```bash
# 开发模式
npm run dev

# 生产模式
npm start
```

## 使用说明

1. 用户认证
- 注册需要有效邀请码（联系管理员获取）
- 密码需满足安全要求：
  - 长度至少8个字符
  - 包含大小写字母、数字和特殊字符
- 登录后可访问所有功能
- 会员功能仅限域锦科技会员使用

2. AI助手使用
- 选择对应场景的AI助手
- 查看欢迎语了解使用方法
- 按照指引提供所需信息
- 获取AI智能回复和建议

3. 数据分析
- 支持上传 CSV/Excel 格式数据
- 自动生成可视化图表
- 支持导出分析结果

4. 思维导图
- 支持 Markdown 格式输入
- 实时预览效果
- 可导出为 SVG 格式
